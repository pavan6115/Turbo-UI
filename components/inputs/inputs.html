Inputs

Inputs are used to to take input from the user. They are used in form to collect data and perform some validations on them.

There are 2 basic types of inputs from which we can interact with the user

Below examples illustrate the 2 types on inputs - Input Boxes | Check Buttons Input

Input Boxes

Example → class="txt-input"

Input Validations Boxes

Example → class="txt-input-val input-error"

Check Buttons Input

Example → class="radio-inp" | class="checkbox-inp"
