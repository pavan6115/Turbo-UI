Avatar

Avatar are used to show a profile picture of a person on social website, apps, blogs or even in navigtion bar as well.

Avatar can be of 2 different types - One with picture and other with Text

Below examples illustrate the 4 different types of avatars. To use any of these avatars, simply add class avatar followed by its type. Example → class="avatar avatar-md"

Avatar with Picture

Example → class="avatar avatar-lg"

Avatar with Text

Example → class="avatar-text avatar-md"
