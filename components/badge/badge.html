Badge

Badge is used where we want to show some notification/message count or show on an icon indicating there is some notification.

Badge can be of 2 different types - One with Avatar and other with Icons

Below examples illustrate the 4 different types of badges. To use any of these badges, simply add class badge followed by its type. Example → class="badge-online"

Badge with Avatar

Example → class="avatar" & span class="badge-away"

-

Badge with Icons

Example → class="badge-icons" & span class="notification"
