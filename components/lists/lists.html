Lists

Lists are used to show some information in point-wise or in order-wise. Mostly used in E-Com sites, stacked notifications, sidebar, navigation bar, etc

There are 2 different types of lists that are used - Simple List | Stacked List

The below examples illustrates the different types of lists. For using them add proper class names
